# RailsProject

Простой блог на RoR, поддерживающий:

* Регистрацию пользователей

* Авторизацию пользователей

* Добавление постов

* Редактирование и удаление постов(только для авторов)

* Добавление комментариев

* Отображение информации о пользователе
